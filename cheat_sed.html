Cheatsheet for sed-the commandline/vim stream editor

Command line options

Syntax: `sed [options] sed-command [input-file]

Option Description Example
-n Suppress defaul pattern space printing sed -n '3 p' employee.txt
-i Backu and modify input file directly sed -ibak 's/John/Johnny/' employee.txt
-f Execute sed script file sed -f scrip.sed employee.txt
-e Execute multiple commands sed -e 'command1' -e 'command2' input-file
